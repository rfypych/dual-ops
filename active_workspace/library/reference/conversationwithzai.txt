1. Buka Browser
Buka Chrome/Firefox, akses:

text

https://connect.8x8.com/
2. Login dengan Akun Lu
Pakai akun yang udah lu daftarin tadi.

3. Buka DevTools
Tekan F12 atau Right Click → Inspect

4. Cari Access Token
Ada 3 cara:

Cara A: Network Tab (Paling Mudah)
Buka tab Network
Refresh halaman (F5)
Klik salah satu request ke connect.8x8.com/api/...
Scroll ke Request Headers
Cari: Authorization: Bearer <token> atau X-Access-Token
Cara B: Application Tab
Buka tab Application
Kiri: Local Storage → https://connect.8x8.com
Cari key seperti: access_token, token, auth, jwt
Atau cek Session Storage
Cara C: Console
Buka tab Console
Ketik:
javascript

localStorage.getItem('access_token')
// atau
document.cookie
